Detailed Description

Class with debugging action defined.

The class consists of pyqt functions to create a window with few push button to create signals which is connected to a slot. When a push button is pressed the do_action is connected. Action window is incorporated with QPushButtons for operations like testing motors, testing heater etc.. When an action is selected the doAction() method is called where the task to be done is taken from the name of the pushbutton, using sender() pyqt method. The action is stored in variable ‘act’ and with if else if loop the condition is checked and the action is performed.

◆ widgets()

def action.action.widgets (   self)

Method to initialize the widgets.

This method creates pushbutton dedicated action as text on it.

  1. Test Motor: To test the working of the motor
  2. Full Operation: Test all the operation in the slave including heater and fan
  3. Get Temp : Get the temperature sensor reading
  4. Get Encoder: Test the encoder
  5. Heater on and Heater off
  6. Test TOF
  7. Open Door: To test the door
  8. init_box :TO initialize a single slave box
  9. Initialize ; To initialise the whole rack
  10. Cancel: To close the action window
