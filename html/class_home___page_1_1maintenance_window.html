Detailed Description

Class maintenanceWindow is used to create a window with an image on the GUi when the mqtt 'Onmaintenance' topic is published by the broker.

When a Onmaintenance topic is published, this class pulls the rack to the maintenance mode. When the rack is maintenance mode all the function in the rack is shutdown except the mqtt communication. When an object for this class is created the init method is automatically called which creates a screen with maintenance image. Until a activate topic arrived the rack is maintained at maintenance mode.
