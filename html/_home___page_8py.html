Detailed Description

This script deals with the basic functioning of the rack. This code is made to autorun whenever the rack boots up.

Version
1.0

This python script is the main GUI window code. Main parts of this scripts are:

  1. Mqtt Client methods
  2. Home Window
  3. Maintenance Window
  4. Offline Window
  5. Rack Configuration thread
  6. Rack initialization thread
  7. Round Robin Thread for heater
  8. Mailing LogFile Function

Mqtt Client methods and main window screen programming are defined in the class Home_Page.Home_Window.
Maintenance Window definition is in class Home_Page.maintenanceWindow.
Offline Window definition is in class Home_Page.offlineWindow.
Rack Configuration and Initialisation are defined in Home_Page.ConfigWorker and Home_Page.intialize_rack classes.
Round Robin algorithm is deployed in the class Home_Page.roundRobinFunc.
Mailing Logfile is defined in the class Home_Page.mail_logfile.
