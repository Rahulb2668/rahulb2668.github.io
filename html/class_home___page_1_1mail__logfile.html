Detailed Description

Class mail_logfile is a child thread class which is invoked from the main.

Whenever there is a request for the log of the rack, the broker publishes a mqtt topic 'maillog' to the rack. When the topic is received, master call the mail_logfile to send the logfile to the email id mentioned. smtplib python library is used to send the email.
