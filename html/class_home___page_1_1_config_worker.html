Detailed Description

A thread for configuring the slave boxes.

ConfigWokker class is used to configure the slave boxes after food is loaded and unloaded. This class call the configure_rack method which is defined in the serialwithpacketisation script. configure_rack is passed a list of boxes to be configured as attribute.As soon as the configure_rack method is finished sending commands of configuration a finishedsignal (pyqtsignal) is emitted. As soon as the food is loaded the temperature and the food type data is send to the slave box as the configuration data.
