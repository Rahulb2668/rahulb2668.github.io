Detailed Description

Function to roundRobin the heaters.

In the rack only 5 heaters are allowed to turn on simultaneously considering the safety of the system. This Qthread class controls the heater after the loading process is done. It collects all the loaded boxes from the database and stores in a list. It checks the temperature of the box at present and compare it with the food temperature that is stored in the database for all the boxes. The difference between the current temperature and required temperature is stored in the list. And accordingly the first 4 boxes with most difference is allowed to heat fisrt. Then after 5 mins this is again invoked and all this procedure is repeated.This thread is started with a QTimer from the main class Home_Page.Home_Window.rtimer.

def Home_Page.roundRobinFunc.run (   self)

This method gets the sorted list of boxes and food temp dictionary from rrchecker method defined in the Database.py If the sorted list is not empty then procedure of round robin is done. For the boxes where the difference is high command to turn on is send and all other is turned off.
Command for heater on is '({ box_id,6,temperature}) '
Command for heater off is '({ box_id,6,0}) '
