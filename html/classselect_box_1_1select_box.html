Detailed Description

Class to create a window to select box for testing.

A window with 23 pushbutton is created when settings topic is published on the broker or masterpwd is entered via keypad These 23 pushbutton represents the slave boxes in the rack. Selecting a box takes to action window where testing actions are performed.
