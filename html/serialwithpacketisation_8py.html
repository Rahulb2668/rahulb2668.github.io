Detailed Description

All method for Serial communication is defined in this script.

Author
: Rahul B

This python script deals with the serial communcaition with the master and the slave. Python have an inbuilt library pyserial which is deployed in the script. Communication used UART. In the rack each of the slave and the master is connected to each other via a RS485 bus. From the master the data is sent to a master controller(Mispigy) which checks the LRC and send the data to bus via UART- RS485 bridge. TX,RX is changes into A,B,Y,Z lines. The RS485 cable have 6 lines 12v, GND, and four data lines A,B,Y,Z. At each slave boxes the same uart-rs485 bridge converts the signal from rs485 to tx and rx and sent to the slave controller
