Detailed Description

Loading_Window class is used to create a pyqt window.

When a loading operation is happening a screen with the food names and slots in which it should be placed is shown on the HMI. In parallel the boxes are opened one by one ready to be loaded.

Member Function Documentation

◆ des()

def Loading_Window.Loading_Window.des (   self)

Method to close the loading Window.

As soon as the finished signal is emitted this function is invoked which call for the configuration of the slots and the updation of the broker. This method closes the loading window after 10000 seconds

◆ Layouts()

def Loading_Window.Loading_Window.Layouts (   self)

This method create a layout for the window.

This method creates the label to display the food item and the box to place text.A main layout is created which consist of a frame widget and a label. This frame widget layout is placefoodlayout which consists of the labels with the text stating the food name and the slot number. As the layouts are created the Qthread class is started to command the slave to open.
